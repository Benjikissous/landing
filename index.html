Une formation de développeur web immersive

Notre formation de développeur web vous apprend toute la stack web, de la base de données au navigateur. Vous apprendrez les concepts de la programmation orientée objet, ainsi qu'à structurer votre logiciel suivant des design patterns comme MVC. En plus de maîtriser les langages de programmation back-end (Ruby, SQL), vous développerez vos compétences de front-end en utilisant les techniques récentes comme flexbox ou CSS grid et vous apprendrez ES6, la dernière version de JavaScript.

Complétez votre éducation

Beaucoup de jeunes diplômés ont l'impression de manquer des compétences nécessaires pour avoir une vie professionnelle stimulante. La formation FullStack du Wagon vous permet d'acquérir les compétences pratiques qui vous permettront de créer votre propre projet, ou de rejoindre les équipes techniques des meilleures startups.
